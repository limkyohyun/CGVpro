# CGVpro

언어 및 라이브러리 : JAVA, Jsp,JavaScript & jQuery, CSS

프레임워크 : Spring, myBatis

DB : Oracle


프로젝트 특징 :

1) 사용자가 이용하는 영화를 웹 사이트로 구현
2) 무비차트, CRUD, 상세페이지, 평점 남기기
3) 이벤트 게시판
4) 로그인, 회원가입, 마이페이지
5) 극장 안내
6) 영화 예매
